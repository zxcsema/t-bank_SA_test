# Обработка ошибок

| Ситуация | Как определяется | Действие системы | Результат для пользователя |
|---|---|---|---|
| ML вернул HTTP 500 | HTTP-статус `5xx` от `ML Recommendation Service` | Зафиксировать `ML_HTTP_5XX`, запустить fallback | `200` с `source = FALLBACK`, если fallback успешен |
| ML недоступен по сети | Ошибка соединения, DNS, отказ подключения | Зафиксировать `ML_NETWORK_ERROR`, запустить fallback | `200` с `source = FALLBACK`, если fallback успешен |
| ML отвечает дольше 3 секунд | Истек тайм-аут ожидания ML | Прекратить ожидание, зафиксировать `ML_TIMEOUT`, запустить fallback | `200` с `source = FALLBACK`, если fallback успешен |
| ML вернул некорректный JSON | Ошибка парсинга или нарушен контракт ответа | Зафиксировать `ML_INVALID_RESPONSE`, запустить fallback | `200` с `source = FALLBACK`, если fallback успешен |
| `confidence` ниже порога | `confidence < threshold` | Зафиксировать `LOW_CONFIDENCE`, запустить fallback | `200` с `source = FALLBACK` |
| ML вернул пустой список | `items` отсутствует или пустой | Зафиксировать `ML_EMPTY_RESULT`, запустить fallback | `200` с `source = FALLBACK`, если fallback нашел объекты |
| User Profile Service недоступен | Ошибка HTTP 5xx, сеть или тайм-аут | Зафиксировать частичную деградацию, продолжить без профиля | `200`, если доступны история или популярность |
| User History Service недоступен | Ошибка HTTP 5xx, сеть или тайм-аут | Зафиксировать частичную деградацию, продолжить без истории | `200`, если доступны профиль или популярность |
| Popularity Service недоступен | Ошибка HTTP 5xx, сеть или тайм-аут | Продолжить по профилю и истории, если они доступны | `200`, если найден результат; иначе `503` |
| Резервный алгоритм не смог сформировать рекомендации | После фильтрации нет объектов или недоступны все внутренние источники | Зафиксировать `FALLBACK_FAILED`, вернуть контролируемую ошибку | `503` с кодом `RECOMMENDATIONS_UNAVAILABLE` |

## Правила частичной деградации

| Недоступные данные | Как формируется результат |
|---|---|
| История | Использовать профиль, параметры поездки и популярность. |
| Профиль | Использовать историю, параметры поездки и популярность. |
| Профиль и история | Использовать параметры поездки и популярность. |
| Популярность | Использовать профиль и историю, если они доступны. |
| Все внутренние источники | Вернуть `503` с контролируемой ошибкой. |

## Пользовательские сообщения

Пользователю не возвращаются сообщения вида "ML-сервис недоступен" или "модель вернула низкий confidence". При успешном fallback пользователь получает обычный список рекомендаций.

При невозможности сформировать рекомендации возвращается нейтральное сообщение:

```json
{
  "code": "RECOMMENDATIONS_UNAVAILABLE",
  "message": "Сейчас не удалось сформировать рекомендации. Попробуйте позже",
  "traceId": "5dc37a1b-5917-4571-971a-a8a3dd111111"
}
```
